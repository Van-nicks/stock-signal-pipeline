# Real-Time Market Data Platform - Streaming Data Engineering System

A production-grade Lambda Architecture platform for real-time market data processing across equities and cryptocurrencies. Streaming pipelines built with Kafka and Spark Structured Streaming deliver trading signals with sub-5-second latency, with batch processing orchestrated through Airflow, Snowflake, and dbt — all deployed on AWS with Terraform infrastructure-as-code, GitHub Actions CI/CD, and Grafana observability.
